Specify the parameters needed by the execution definition ids, when they are running on your SilkCentral Test Manager system.
Insert them in the "name=value" format. You can enter more than one parameter by using a comma (or newline) separated list of parameters. E.g.: "param1=value,param2=another value" or:
param1=value
param2=another value

Note: The Parameter name is case sensitive and the parameters are set on all execution definitions specified on your configuration field "Execution Definition ID(s)".
