Geben Sie eine bestehende Silk Central Testsuite-ID ein, oder mehrere IDs als Komma-getrennte Liste (z.B. 12345,3389). Sämtliche Testuite-IDs müssen zum gleichen Projekt gehören, ansonsten benötigen Sie zusätzliche Builder-Konfigurationen. Die Reihenfolge der eingegebenen Testsuite-IDs hat keinen Einfluss auf die tatsächliche Ausführungsreihenfolge. Falls Sie eine bestimmte Reihenfolge brauchen, benötigen Sie zusätzliche Builder-Konfigurationen. Die Testsuiten werden parallel ausgeführt, falls die Silk Central Konfiguration dies erlaubt. Weitere Informationen hierzu finden Sie unter 'Testumgebung (Seite)' in der Silk Central Hilfe.
